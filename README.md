A Python utility for amateur radio operators that polls the CYYZ (Toronto Pearson) METAR feed every 5 minutes and keeps a CSNSAt controlled rotator aligned with the prevailing wind direction. Commands are sent to a SAT or PSTRotator-compatible controller over UDP. The script is "polite" — it monitors port 12001 for positioning commands from other applications (satellite trackers, DX clusters, etc.) and automatically yields control for 5 minutes after any external command is detected. The antenna is only repositioned when wind speed exceeds a configurable threshold (default 13 kt). 
Users need to edit the Python file for their IP addresses and ICAO weather station
